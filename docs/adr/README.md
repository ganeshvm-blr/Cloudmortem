# CloudMortem Architecture Decision Records

Architecture Decision Records (ADRs) document important technical decisions,
the reasoning behind them, alternatives considered, and long-term consequences.

## ADR Status Definitions

### Proposed

A decision has been identified but implementation has not been completed.

### Accepted

The decision has been implemented and is the current architecture.

### Deprecated

The decision has been replaced by a newer approach.
